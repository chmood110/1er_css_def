Programación Ladder para PLC

Esta página muestra una introducción sencilla a la lógica ladder, explicando qué es, para qué sirve y algunos ejemplos básicos.

¿Qué es la programación ladder?

La programación ladder es una forma de programar controladores lógicos programables usando diagramas que se parecen a un circuito eléctrico. Se utiliza mucho en la automatización industrial porque resulta visual y práctica para representar procesos.

Contacto normalmente abierto

Se activa cuando recibe una señal. Es uno de los elementos más usados al empezar a trabajar con ladder.

Bobina

Representa una salida. Cuando la lógica de la línea se cumple, la bobina se energiza.

Temporizador

Permite retrasar acciones dentro de un proceso automático, por ejemplo encender un motor después de algunos segundos.

Ejemplo básico

En un circuito ladder sencillo, un pulsador puede activar una salida como una lámpara. Esto permite entender la relación entre entradas, condiciones y salidas dentro de un PLC.

Conclusión
