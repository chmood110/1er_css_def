Introducción a Flexbox

En esta práctica se muestran ejemplos del uso de Flexbox en CSS para organizar y alinear elementos.

¿Qué es Flexbox?

Flexbox es un sistema de diseño en CSS que permite organizar elementos de forma flexible dentro de un contenedor. Es útil para alinear, distribuir y adaptar elementos horizontal o verticalmente.
