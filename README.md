# Estilos Responsivos — Programación Ladder para PLC

Sitio web informativo desarrollado con **HTML y CSS**, enfocado en la introducción a la **programación Ladder para PLC** dentro del contexto de la automatización industrial.  
La página presenta una explicación clara y visual sobre los fundamentos de los **Controladores Lógicos Programables (PLC)**, la lógica Ladder, sus elementos básicos, ejemplos de funcionamiento y aplicaciones comunes en la industria.

## Descripción

Este proyecto fue creado como una página web estática con diseño responsivo, utilizando únicamente tecnologías base del desarrollo frontend.  
Su objetivo es ofrecer una introducción visual y técnica al lenguaje Ladder, ampliamente utilizado en sistemas de control industrial.

La interfaz incluye una estructura organizada por secciones, con un enfoque visual limpio, navegación clara y adaptación a diferentes tamaños de pantalla mediante **media queries**.

## Contenido de la página

La página incluye las siguientes secciones:

- Barra de navegación
- Sección principal de presentación
- Introducción a los PLC
- Explicación de la lógica Ladder
- Elementos básicos del lenguaje Ladder
- Ejemplo visual
- Flujo básico de trabajo de un PLC
- Ejemplo de arranque y paro
- Señales de entrada y salida
- Aplicaciones industriales
- Footer informativo
- Diseño responsivo
